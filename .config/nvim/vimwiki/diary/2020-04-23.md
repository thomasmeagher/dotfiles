Useful tmux commands (prefix omitted):

| Command       | Description             |
| ------------- | ----------------------- |
| `z`           | Zoom pane               |
| `f`           | Find window             |
| `w`           | List windows            |
| `&`           | Kill window             |
| `$`           | Rename session          |
| `x`           | Kill pane               |
| `,`           | Rename window           |
| `/`           | Search                  |
| `!`           | Move pane to new window |
